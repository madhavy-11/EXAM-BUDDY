# 📚 Exam Buddy

A professional exam schedule management system built with Python and SQLite. Perfect for students to track their exams, get reminders, and stay organized.

---

## ✨ Features

### Core Features
- ✅ **Add Exams** - Add exams with subject, date, time, location, and teacher
- ✅ **View Exams** - View all exams in a clean table format
- ✅ **Edit Exams** - Update exam details anytime
- ✅ **Delete Exams** - Remove exams with confirmation
- ✅ **Search Exams** - Search by subject or teacher name

### Smart Features
- 📅 **Upcoming Exams** - View all future exams
- ⏰ **Past Exams** - View completed exams
- 🎯 **Today's Exams** - See what's scheduled for today
- 📆 **Next 7 Days** - Plan for the week ahead
- 📊 **Statistics** - See exam distribution and insights

### Data Management
- 💾 **SQLite Database** - Professional, fast, and reliable storage
- 💿 **Backup System** - Automatic database backups
- 📤 **Export to JSON** - Export your schedule anytime
- 🔄 **Data Migration** - Import from old JSON files

### Coming Soon
- 📧 **Email Reminders** - Get notifications via email
- 🌐 **Web Interface** - Access from any browser
- 📱 **Mobile App** - Manage exams on the go
